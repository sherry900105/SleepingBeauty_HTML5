This is an endless running game. The objective of the game is to keep jumping to get higher score. 
Player can control the 'Doodle' by pressing 'left' and 'right' button. Player should try to keep jumping on the bar, and avoid the bombs.
If the player hits a bomb, he will lose a life. And when he uses up the 5 lives, game is over. Also, if the player falls from the lowest bar on the screen, game will be over too.
